# Pantone C Colors now in the Color Tools

I'm happy to share I've added a large batch of [Pantone](https://www.pantone.com/) C colors to my various quilty color tools.
I learned a little bit about the Pantone history from [Wikipedia](https://en.wikipedia.org/wiki/Pantone), too.
It's interesting!

In the quilting industry, we don't have any solids lines that have nearly as many colors as Pantone has - Kona gets the closest.
However, if you are thinking about digital design only, it's nice to play around with these swatches.

Enjoy the tools linked below!

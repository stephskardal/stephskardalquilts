# Welcome to "A Quilty Solid"

Hi there! I've just released my new iOS app called "A Quilty Solid",
which ports over a few of my existing web tools to a more mobile experience,
and adds a few new features.

The tools version 0.1 launches with are:

- A color list for each of the major solids manufacturers, with color detail per swatch.
- A Pantone matcher which matches closest colors to each Pantone color.
- A color palette tool which helps find solids colors matching to your uploaded photo.

All tools include an additional brief help screen for more information.
I expect to add more tools over time, as well as improve existing tools.
